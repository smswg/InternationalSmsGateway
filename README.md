# SMSWG国际短信运营综合管理平台

## 系统介绍

​           SMSWG国际短信网关是由合肥标通科技自主研发的短信系统,已获得国家软件版权证明,SMSWG国际短信网关支持的协议：SMPP/HTTP协议,系统采用java原生开发,支持集群部署,日发送量可高达1亿条,系统同时支持私有化部署。如果您准备运营自己的国际短信网关系统平台,建议您选择我们，我们有专业的运维团队保障您的业务平台稳定。

系统官网地址https://:www.smswg.com 如您有对接私有短信协议请底部扫码联系。


## 关于性能

           SMSWG国际短信网关部署在服务器16核心32G内存配置，测试每秒可下发1.5万条短信。

## 重点说明

           SMSWG国际短信网关系统目前已经合作了很多客户,系统稳定强悍，让您运营无后顾之忧。 

## 支持国际短信发送-SMPP协议

           SMSWG短信网关系统有国际短信系统版本，支持不同国家不同短信单价，详情请加微信了解。

## 我们的优势

          1.smswg短信网关采用原生java开发,模块化设计,短信发送并发更强。
          2.平台功能齐全,PC管理端+PC用户端+PC代理商端,上手即可运营,运营方便快捷。
          3.平台支持功能二次开发,支持更多的自定义业务需求。
          4.smswg短信网关支持发送国际短信，同时支持根据发送国家设置短信单价等功能。     
          5.smswg短信网关支持永久买断模式+源代码。
